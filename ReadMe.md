<h2>How to install Tkinter</h2>
<br>

With these commands you can install Tkinter and run any Program using the Tkinter library from the Terminal

<b>1. macOS</b>
<ul>
  <li>brew install python-tk</li>
</ul>

<b>2. Linux (Arch)</b>
<ul>
  <li>sudo pacman -S tk</li>
</ul>
